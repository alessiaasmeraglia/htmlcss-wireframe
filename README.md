# htmlcss-wireframe

Esercizio: riprodurre un wireframe in **HTML e CSS** concentrandosi su **box model** (margin, padding, border).

## Obiettivo
- Ricreare il layout dello screenshot (header, card, footer)
- Spaziature coerenti e armoniose tramite box model
- Card con altezza base uguale, ma che crescono se il testo è maggiore

## Struttura del progetto
- index.html
- style.css
- .gitignore

## Note
- Layout impostato in modo **mobile-first**
